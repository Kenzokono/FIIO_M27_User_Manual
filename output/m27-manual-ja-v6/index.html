外観ガイド — ボタン・端子の説明

上面

  • 3.5mmヘッドホン出力 / ライン出力 — シングルエンドのヘッドホン出力端子です。ライン出力としても使用できます。
  • 4.4mmバランスヘッドホン出力 / バランスライン出力 — バランス接続用のヘッドホン出力端子です。バランスライン出力としても使用できます。
  • 6.35mmヘッドホン出力 / ライン出力 — デスクトップモード時に使用する標準サイズのヘッドホン端子です。ライン出力としても使用できます。
  • 電源/ボリュームノブ — 回転して音量を調整します。押すと画面のオン/オフを切り替えます。長押しで電源オン/オフを行います。

左側面

  • マルチファンクションボタン — 短押しでFiiO Musicアプリに切り替わります。長押しでモード選択メニューを表示します。Settings（設定）→ Global（グローバル）→ Multifunction button から機能を変更できます。
  • 前の曲ボタン — 短押しで前のトラックに戻ります。長押しで巻き戻しします。
  • 再生/一時停止ボタン — 再生と一時停止を切り替えます。
  • 次の曲ボタン — 短押しで次のトラックに進みます。長押しで早送りします。

右側面

  • アンテナ — Wi-FiおよびBluetoothの受信感度を向上させます。アンテナ部分を金属物で覆わないでください。ワイヤレス性能に影響します。通常の手持ち操作では信号受信に影響はありません。
  • アンビエントライト — 側面のLEDライトです。再生状態やサンプリングレートに応じて色が変わります。

底面

  • 光（オプティカル）出力 — 光デジタル信号を出力します。
  • USB Type-C（充電/電源）POWER IN — 充電および電源供給に使用します。PD2.0/3.0急速充電のみ対応します。その他の充電器では通常充電となります。
  • HOLDスイッチ — 物理ボタンおよびタッチスクリーンの誤操作を防止するロックスイッチです。
  • USB Type-C（データ） — データ転送、USB DACモード、USB OTG接続に使用します。
  • MicroSDカードスロット — 最大2TBのmicroSDカードに対応します。2枚同時に挿入して使用できます。
  • USB HOST — USB DACやUSBメモリなどの外部USBデバイスを接続します。コンピューター接続や充電には対応しません。
  • 同軸（コアキシャル）入出力（COAX IN/OUT） — 同軸デジタル信号の入力・出力に使用します。
  • デスクトップモードスイッチ — デスクトップモードのオン/オフを物理的に切り替えます。USB接続時にデスクトップモードが有効になります。

USB Type-Cポートの機能比較

M27には2つのUSB Type-Cポートがあります。用途が異なるため、接続時にご注意ください。

機能 USB Type-C（データ） USB Type-C（充電/電源）
充電 対応（通常充電） 対応（QC4+/PD3.0）
データ転送 対応 非対応
USB DACモード 対応 非対応
USB OTG 対応 非対応
DC電源入力（デスクトップモード） 非対応 対応

出力端子の仕様

M27の3.5mm、4.4mm、6.35mmのアナログ出力は同時に出力可能です。出力の優先順位は以下のとおりです。

USB出力 ＞ Bluetooth ＞ 同軸 ＝ 光 ＝ 3.5mm・4.4mm・6.35mmアナログ出力

ただし、複数のポートを同時に接続すると電力損失が生じる可能性があります。一度に1つのポートのみの接続を推奨します。

デスクトップモード時は6.35mm出力が最優先で使用されます。

注意：ライン出力として使用する場合は、Settings（設定）→ Audio（オーディオ）で出力モードを切り替えてください。Line Out（ライン出力）を選択します。

デスクトップモードスイッチの詳細

底面のデスクトップモードスイッチでポータブルモードとデスクトップモードを切り替えます。各モードで使用できる機能は以下のとおりです。

機能 ポータブルモード デスクトップモード
3.5mmヘッドホン出力 対応 対応
4.4mmバランス出力 対応 対応
6.35mmヘッドホン出力 / ライン出力 非対応 対応
USB HOST 非対応 対応
COAX IN（同軸入力） 非対応 対応
COAX OUT（同軸出力） 対応 対応
バッテリー駆動 対応 非対応（常時電源接続が必要）

製品概要

この度はFiiO M27をお買い上げいただき、ありがとうございます。本マニュアルがM27の可能性を最大限に引き出す一助となれば幸いです。

FiiO M27はフラッグシップクラスのAndroidベースデジタルオーディオプレーヤー（DAP）です。ESS ES9038PRO D/Aコンバーター（DAC）を2基、THX AAA-788+ヘッドホンアンプを2基搭載しています。

ポータブルモードとデスクトップモードの2つの使い方ができます。デスクトップモードでは6.35mmヘッドホン出力やUSB HOST、同軸入力が利用可能になります。

主な特長

  • ESS ES9038PRO × 2基によるデュアルDAC構成
  • THX AAA-788+ × 2基によるヘッドホンアンプ
  • 4.4mmバランス出力で最大1500mW+1500mW（32Ω負荷時）
  • 5.0インチIPS液晶（1080×1920）
  • Qualcomm Snapdragon 665プロセッサー
  • 4GB RAM / 64GB内部ストレージ
  • 9800mAh大容量バッテリー（最大約15時間再生）
  • Bluetooth 5.1（LDAC / aptX Adaptive / aptX HD対応）
  • Roon Ready / AirPlay / DLNA対応
  • MQA 8×アンフォールド対応

付属品

  • FiiO M27本体
  • USB Type-Cケーブル
  • 保護フィルム（貼付済み）
  • クイックスタートガイド
  • 保証書

クイックスタート

充電

初めて使用する際は、十分に充電してください。底面のUSB Type-C（充電/電源）ポートに充電器を接続します。

  • QC4+ / PD3.0対応の充電器で最大27Wの急速充電が可能です。
  • フル充電まで約3時間です（急速充電器使用時）。

電源オン/オフ

  1. 電源オン — 電源/ボリュームノブを長押しします。FiiOのロゴが表示されたら起動中です。
  2. 電源オフ — 電源/ボリュームノブを長押しし、表示されたメニューから「Power off」（電源オフ）を選択します。
  3. 画面のオン/オフ — 電源/ボリュームノブを短く押します。

microSDカードの挿入

  1. 底面のmicroSDカードスロットに、microSDカードをカチッと音がするまで差し込みます。
  2. M27が自動的にカードを認識します。

注意：microSDカードは電源オフ時に挿入・取り外しすることを推奨します。使用中に取り外す場合はSettings（設定）→ Storage（ストレージ）から安全な取り外しを実行してください。

ステータスバーと表示画面

画面上部のステータスバーには、M27の動作状態が表示されます。

画面上部から下にスワイプすると、クイック設定パネルが表示されます。以下の機能に素早くアクセスできます。

  • Internet — Wi-Fiのオン/オフを切り替えます。
  • Bluetooth — Bluetoothのオン/オフを切り替えます。
  • Mode Choose（モード選択） — 6つの動作モードを切り替えます。
  • Gain（ゲイン） — ヘッドホン出力のゲインをLow / High / Ultra Highの3段階で切り替えます。
  • PO — 3.5mmヘッドホン出力をオン/オフします。
  • BAL.PO — 4.4mmバランスヘッドホン出力をオン/オフします。
  • OPT/COAX — 光/同軸デジタル出力をオン/オフします。
  • Auto-rotate（自動回転） — 画面の自動回転を切り替えます。

6つの動作モード

FiiO M27には6つの独立した動作モードがあります。Mode Choose（モード選択）から切り替えできます。

Android Mode（Androidモード）

標準的な動作モードです。すべての機能とサードパーティアプリが利用できます。ストリーミングサービスやファイル管理など、フル機能のAndroid環境として使用します。

Pure Music Mode（Pure Musicモード）

音楽再生に特化したモードです。不要なバックグラウンドプロセスを停止し、オーディオ品質を最大限に高めます。FiiO Musicアプリのみが動作します。

Pure Musicモードではバッテリー駆動時間も延長されます。

AirPlay Mode（AirPlayモード）

M27をAirPlayレシーバーとして動作させるモードです。iPhone、iPad、Macからワイヤレスで音楽を送信して再生できます。

  1. M27とAppleデバイスを同じWi-Fiネットワークに接続します。
  2. Mode ChooseからAirPlayモードを選択します。
  3. Appleデバイスのコントロールセンターで、AirPlay出力先として「FiiO M27」を選択します。
  4. Appleデバイスで音楽を再生すると、M27から音が出力されます。

USB DAC Mode（USB DACモード）

M27をUSB DACとして使用するモードです。パソコンやスマートフォンに接続して、M27の高品質なDACとアンプで音楽を再生します。

Windowsの場合

  1. Mode ChooseからUSB DAC Modeを選択します。
  2. 底面のUSB Type-C（データ）ポートでパソコンに接続します。
  3. FiiO USB DACドライバーをインストールします（初回のみ）。
  4. Windowsのサウンド設定で「FiiO M27」を出力デバイスとして選択します。

macOS / Linuxの場合

  1. Mode ChooseからUSB DAC Modeを選択します。
  2. 底面のUSB Type-C（データ）ポートでパソコンに接続します。
  3. システムのサウンド設定で「FiiO M27」を出力デバイスとして選択します。

macOSおよびLinuxではドライバーのインストールは不要です。

対応フォーマット（USB DACモード）

  • PCM: 最大384kHz/32bit
  • DSD: DSD64/128/256（ネイティブ / DoP）
  • MQA対応

Bluetooth Receiving Mode（Bluetooth受信モード）

M27をBluetoothレシーバーとして使用するモードです。スマートフォンなどからBluetoothで音楽を受信して再生します。

  1. Mode ChooseからBluetooth Receiving Modeを選択します。
  2. スマートフォンのBluetooth設定で「FiiO M27」を検索してペアリングします。
  3. スマートフォンで音楽を再生すると、M27から音が出力されます。

対応Bluetoothコーデック（受信）: SBC、AAC、LDAC

ペアリングモードへの入り方

  1. 初めてBluetooth受信モードに切り替えた時、自動的にペアリングモードに入ります。
  2. 再生/一時停止ボタンを長押しすると、強制的にペアリングモードに入ります。
  3. 接続中のスマートフォンからBluetooth接続を切断すると、ペアリングモードに入ります。

ペアリング情報の消去

方法1: Bluetooth受信モードページ → Settings（設定）→「Clear Pairing（ペアリング消去）」をタップします。

方法2: 次の曲ボタンを押します。

注意：M27は過去にペアリングしたデバイスとの自動再接続に対応しています。再生/一時停止ボタンを短く押すことで再接続できます。Bluetooth接続時、M27のボタンで再生/一時停止および曲送り操作が可能です。

COAX Mode（COAXモード）

M27を同軸DACとして使用するモードです。外部機器から同軸デジタル信号を入力し、M27のDACとアンプで再生します。

  1. Mode ChooseからCOAX modeを選択します。
  2. 底面の同軸入力端子に同軸デジタルケーブルを接続します。
  3. 外部機器で再生を開始すると、M27から音が出力されます。

よく使う機能の説明

スクリーンショット

M27でスクリーンショットを撮影できます。

操作方法: 3本指で画面を下にスワイプします。または、電源ボタンと音量-ボタンを同時に押します。画面が一瞬フラッシュし、スクリーンショットが保存されます。

保存先は内部ストレージのPictures/Screenshotsフォルダーです。

USBオーディオデジタル出力

M27のUSB HOST端子から外部USB DACにデジタルオーディオ信号を出力できます。詳しい手順は「USBオーディオ出力」を参照してください。

FiiO Link — Bluetooth制御

FiiO Linkを使用すると、スマートフォンからBluetooth経由でM27をリモートコントロールできます。

  1. M27のFiiO Musicアプリ → Settings（設定）→ FiiO Link → ┆ → 「Bluetooth as Server」を選択します。
  2. M27とスマートフォンをBluetoothで接続します。
  3. スマートフォンのFiiO Musicアプリ → Settings（設定）→ FiiO Link → ┆ → 「Bluetooth as Client」を選択します。
  4. 接続後、スマートフォンからM27の音楽ライブラリの閲覧や再生コントロールができます。

注意：iPhoneはFiiO Link Wi-Fi制御のみ対応しています。Bluetooth制御には対応していません。

FiiO Link — Wi-Fi制御

Wi-Fi経由でFiiO Linkを使用することもできます。Bluetoothよりも安定した接続で操作できます。

  1. M27とスマートフォンを同じWi-Fiネットワークに接続します。
  2. M27のFiiO Musicアプリ → Settings（設定）→ FiiO Link → ┆ → 「WiFi as Server」を選択します。IPアドレスが表示されます。
  3. スマートフォンのFiiO Musicアプリ → Settings（設定）→ FiiO Link → ┆ → 「WiFi as Client」を選択し、表示されたIPアドレスを入力します。
  4. 接続後、スマートフォンからM27をリモートコントロールできます。

注意：iPhoneはFiiO Link Wi-Fi制御のみ対応しています。Bluetooth制御には対応していません。

DLNA機能

FiiO M27はDLNA（Digital Living Network Alliance）に対応しています。同じWi-Fiネットワーク上にあるDLNA対応デバイスと音楽ファイルを共有・再生できます。

M27をDLNAレンダラーとして使用する

パソコンやNASなどのDLNAサーバーから音楽をM27に送信して再生できます。

  1. M27とDLNAサーバーを同じWi-Fiネットワークに接続します。
  2. FiiO Musicアプリを起動します。
  3. DLNAコントローラーアプリ（パソコンのメディアプレーヤーなど）からM27を出力先として選択します。
  4. DLNAコントローラーで音楽を選択して再生します。M27から音が出力されます。

M27でDLNAサーバーの音楽をブラウズする

M27からDLNAサーバー上の音楽ファイルをブラウズして再生できます。

  1. M27とDLNAサーバーを同じWi-Fiネットワークに接続します。
  2. FiiO Musicアプリのメニューから「DLNA」を選択します。
  3. 表示されたDLNAサーバーを選択し、音楽ファイルをブラウズします。
  4. 再生したい曲を選択します。

注意：DLNA機能を使用できないデバイスがある場合は、support@fiio.com までご連絡ください。FiiO Musicアプリは現在、音楽の受信のみに対応しています。送信機能は今後の追加を検討しています。

Roon Ready

FiiO M27はRoon Readyに対応しています。Roonアプリから高品質なストリーミング再生が可能です。

  1. M27とRoon Coreサーバーを同じネットワークに接続します。
  2. FiiO Musicアプリを起動します。Roon Ready機能は自動的に有効になります。
  3. Roonアプリの Settings（設定）→ Audio（オーディオ）でFiiO M27が表示されます。「Enable」をクリックして有効にします。
  4. Roonアプリのゾーン選択でM27を出力先として選択します。
  5. Roonアプリから音楽を再生すると、M27から音が出力されます。

注意：Roon Readyを使用するには、別途RoonのサブスクリプションとRoon Coreサーバーが必要です。

光/同軸出力の使い方

M27は光デジタル出力および同軸デジタル出力に対応しています。外部DACやアンプに接続する際に使用します。

光デジタル出力

  1. 底面の光出力端子に光デジタルケーブルを接続します。
  2. Settings（設定）→ Audio（オーディオ）→ Optical/Coaxial output（光/同軸出力）をオンにします。
  3. FiiO Musicで音楽を再生すると、光デジタル信号が出力されます。

同軸デジタル出力

  1. 底面の同軸出力端子に同軸デジタルケーブルを接続します。
  2. Settings（設定）→ Audio（オーディオ）→ Optical/Coaxial output（光/同軸出力）をオンにします。
  3. FiiO Musicで音楽を再生すると、同軸デジタル信号が出力されます。

注意：光/同軸出力をオンにすると、3.5mm端子からのアナログ出力は無効になります。アナログ出力に戻すには、光/同軸出力をオフにしてください。

DSD信号をS/PDIFで伝送する場合、DoP（DSD over PCM）方式を使用します。Settings（設定）→ Audio（オーディオ）でDoPの設定を変更できます。

デスクトップモード

FiiO M27はポータブルモードとデスクトップモードの2つの形態で使用できます。底面のデスクトップモードスイッチを切り替えることで、モードが変わります。

デスクトップモードの特長

  • 6.35mmヘッドホン出力 / ライン出力 — デスクトップモードでのみ使用可能です。高インピーダンスのフルサイズヘッドホンを駆動できます。ライン出力としても使用できます。
  • USB HOST — 外部USB DACやUSBメモリを接続できます。M27をトランスポートとして使用する場合に便利です。
  • COAX IN（同軸入力） — 外部機器から同軸デジタル信号を受信します。
  • 常時電源供給 — USB Type-C（充電/電源）から常時電源を供給しながら使用できます。バッテリー劣化を防ぐバイパス充電機能を搭載しています。

注意：デスクトップモードスイッチの切り替え後、M27が自動的に再起動する場合があります。データの保存を確認してから切り替えてください。

注意：USBからの電力供給が不足している場合、バッテリーから追加の電力が供給されます。バッテリー残量が15%未満の場合、デスクトップモードであっても、USB接続時に強制的にバッテリーが充電されます。

Bluetooth送信設定

M27をBluetoothヘッドホンやBluetoothスピーカーに接続する場合の設定です。

  1. Settings（設定）→ Bluetooth メニューを開き、Bluetoothをオンにします。
  2. 接続先のBluetoothデバイス（イヤホン、スピーカーなど）をペアリングモードにします。
  3. M27のBluetooth設定画面に検出されたデバイスが表示されます。デバイスをタップしてペアリングします。
  4. 「Connected」（接続済み）と表示されたら接続完了です。

対応Bluetoothコーデック（送信）: SBC、AAC、aptX、aptX HD、aptX Adaptive、LDAC

注意：Bluetoothコーデックは接続先のデバイスが対応しているコーデックに自動的に切り替わります。高音質コーデックを使用するには、接続先のデバイスも同じコーデックに対応している必要があります。

USBオーディオ出力

M27のUSB HOST端子に外部USB DACを接続して、デジタルオーディオ信号を出力できます。M27をトランスポート（デジタル音源送出装置）として使用する場合に有効です。

  1. デスクトップモードスイッチをオンにします。
  2. 底面のUSB HOST端子に外部USB DACを接続します。
  3. Settings（設定）→ Audio（オーディオ）→ USB Audio Output（USBオーディオ出力）をオンにします。
  4. FiiO Musicで音楽を再生すると、外部USB DACから音が出力されます。

注意：FiiO Music以外のサードパーティアプリでは、USBオーディオ出力が正常に動作しない場合があります。ビットパーフェクト出力にはFiiO Musicの使用を推奨します。

ファームウェアアップデート

M27はオンラインアップデートとローカルアップデートの2つの方法でファームウェアを更新できます。

オンラインアップデート（OTA）

  1. M27をWi-Fiに接続します。
  2. Settings（設定）→ About Device（デバイス情報）→ Online Upgrade（オンラインアップグレード）を選択します。
  3. 新しいファームウェアが利用可能な場合、アップデートの案内が表示されます。
  4. 「Download and Install」（ダウンロードしてインストール）を選択して実行します。

アップデートの所要時間は約5〜20分です。アップデート後のシステム再起動に約2分かかります。

注意：オンラインアップデート時に「invalid image」と表示された場合は、内蔵ストレージの android/data/com.android.fiioupdate/files フォルダーを削除し、再度オンラインアップデートをお試しください。

ローカルアップデート

  1. FiiO公式サイトからファームウェアパッケージファイルをダウンロードします。
  2. ファームウェアパッケージファイルをM27の内部ストレージまたはSDカードのルート（最上位フォルダー）にコピーします。
  3. Settings（設定）→ About Device（デバイス情報）→ Local Upgrade（ローカルアップグレード）を選択します。
  4. ファームウェアファイルを選択して、アップデートを実行します。

注意：ファームウェアパッケージファイルは解凍しないでください。ダウンロードしたファイルをそのままコピーしてください。

警告：ファームウェアアップデート中は電源を切らないでください。バッテリー残量が30%以上あることを確認してから実行してください。アップデートの失敗は故障の原因となります。

ファームウェアアップデート中にリカバリーモードが表示され、ファームウェアが検出されない場合があります。インストールパッケージが破損している可能性があります。電源ボタンを押して「reboot」（再起動）を選択し、ファームウェアファイルを再度ダウンロードしてお試しください。

強制再起動

M27がフリーズして操作を受け付けない場合、強制再起動を行います。

操作方法: 「FiiO」ロゴが表示されるまで、電源ボタンを約10秒間長押しします。

注意：強制再起動を行うと、保存されていないデータが失われる可能性があります。通常の再起動ができない場合にのみ使用してください。

リカバリーモード

M27が正常に起動しない場合や、工場出荷時リセットが必要な場合にリカバリーモードを使用します。

操作方法: 電源オフの状態で、電源ボタンとボリュームノブを同時に長押しします。リカバリーモード画面が表示されます。

リカバリーモードでは以下の操作が可能です。

  • Reboot system now（再起動） — 通常モードで再起動します。
  • Wipe data/factory reset（データ消去/工場出荷時リセット） — すべてのユーザーデータを消去し、工場出荷時の状態に戻します。
  • Apply update from storage（ストレージからアップデート適用） — 内部ストレージまたはmicroSDカードに保存されたファームウェアファイルを適用します。

警告：「Wipe data/factory reset」を実行すると、内部ストレージのすべてのデータが消去されます。必要なデータのバックアップを事前に行ってください。

工場出荷時リセットの詳細手順

工場出荷時リセットは以下の2つの方法で実行できます。

方法1: Settings（設定）から実行

  1. Settings（設定）→ System（システム）→ Reset options（リセット）を開きます。
  2. 「Erase all data (factory reset)」（すべてのデータを消去）を選択します。
  3. 確認画面で「ERASE ALL DATA」を選択して実行します。
  4. M27が自動的に再起動し、工場出荷時の状態に戻ります。

方法2: リカバリーモードから実行

  1. 上記の手順でリカバリーモードに入ります。
  2. 「Wipe data/factory reset」を選択します。
  3. 確認画面で「Yes」を選択します。
  4. 完了後、「Reboot system now」を選択して再起動します。

サードパーティアプリのインストール

FiiO M27にはGoogle Playストアが搭載されていません。サードパーティアプリのインストールにはAPKファイルを使用します。

  1. パソコンでAPKファイルをダウンロードします。
  2. M27をUSBケーブルでパソコンに接続します。
  3. パソコンからM27の内部ストレージにAPKファイルをコピーします。
  4. M27のファイルマネージャーアプリでAPKファイルを開き、インストールします。

注意：初回のAPKインストール時に「不明なソースからのインストールを許可しますか？」という確認が表示されます。「許可」を選択してインストールを続行してください。

FiiO独自のアプリストアも利用できます。ホーム画面の「FiiO Market」アプリから、FiiOが検証済みのアプリをダウンロードできます。

システム設定

Settings（設定）アプリから、M27の各種設定を変更できます。設定メニューは以下のカテゴリーで構成されています。

Wi-Fi

Wi-Fiネットワークに接続します。2.4GHz帯と5GHz帯の両方に対応しています。

  • 利用可能なWi-Fiネットワーク一覧から接続先を選択します。
  • パスワード保護されたネットワークの場合は、パスワードを入力して接続します。

注意：「Wi-Fi on during sleep（スリープ中のWi-Fi）」オプションが無効（デフォルト）の場合の動作です。スリープ中にWi-Fiが自動的にオフになります。復帰時に再接続します。消費電力を大幅に削減できます。有効にすると、スリープ中もWi-Fi接続を維持しますが、消費電力は増加します。

Bluetooth

Bluetooth送信の設定を行います。詳細は「Bluetooth送信設定」を参照してください。

More（その他）

VPN、テザリングなどのネットワーク関連の追加設定です。

Audio（オーディオ）

FiiO M27のオーディオ専用設定です。

1. Optical/Coaxial output（光/同軸出力）

光出力または同軸出力を使用する場合は、このスイッチをオンにしてください。

対応フォーマット:

  • 同軸出力 — PCM 384kHzまで、DSD128まで対応
  • 光出力 — PCM 192kHzまで対応

上記を超えるサンプリングレートの音源は、SRC（サンプルレートコンバーター）処理が行われます。

2. Channel balance（チャンネルバランス）

左右チャンネルの出力振幅を個別に調整して、音量バランスを設定します。デフォルト値は0で、各チャンネルは20段階で調整可能です（1段階あたり約0.5dB）。

左右の聴力に差がないユーザーは変更不要です。

3. Gain（ゲイン）

ヘッドホン出力のゲインを5段階で切り替えます。

  • Low（低ゲイン） — 高感度のイヤホンやIEMに適しています。
  • Medium（中ゲイン） — 一般的なイヤホンに適しています。
  • High（高ゲイン） — 一般的なヘッドホンに適しています。
  • Super High（スーパーハイゲイン） — 高インピーダンスのヘッドホンに適しています。
  • Ultra High（ウルトラハイゲイン） — 最も駆動力が必要なヘッドホンに適しています。

注意：Ultra Highゲインはデフォルトでは非表示です。急速充電電源接続時のみ利用可能になります。

警告：ゲインを切り替える際は、一度音量を下げてから行ってください。突然の大音量により聴覚を損傷するおそれがあります。

4. All to DSD

PCM音源をリアルタイムでDSDに変換して再生します。DSD64、DSD128、DSD256から変換先を選択できます。

DSD64は44.1kHz/16bitのPCMと比較して約4倍のデータ量を持ちます。アップサンプリングにより音質向上の効果があります。サードパーティアプリの再生にも適用されます。

注意：All to DSD機能を有効にすると、CPU負荷が増加し、バッテリー消費と本体温度が上昇します。バッテリー駆動時間が公式スペックより短いと感じる場合は、All to DSDの無効化を推奨します。

5. Digital Filter（DACフィルター）

DACのデジタルフィルターを選択できます。音質の微細な違いを楽しめます。

  • Sharp Roll-Off
  • Slow Roll-Off
  • Short Delay Sharp Roll-Off
  • Short Delay Slow Roll-Off
  • Super Slow Roll-Off

6. Volume Steps（ボリュームステップ）

音量調整の段階数を選択します。120段階（細かい調整）または60段階（標準）から選択できます。

7. Bluetooth codec（Bluetoothコーデック）

Bluetooth送信時、デバイスはデフォルトで対応する最高品質のコーデックで接続します。このオプションでは、優先する最高対応コーデックを手動で選択できます。

対応コーデック:

  • LHDC
  • LDAC
  • aptX Adaptive
  • aptX HD
  • aptX
  • AAC
  • SBC

注意：コーデック設定を変更した場合、Bluetooth接続を一度切断してから再接続する必要があります。

8. Bluetooth audio codec LDAC（LDAC再生品質）

LDACコーデック使用時の再生品質モードを切り替えます。

  • 音質優先 — 990kbps / 909kbps
  • バランス — 660kbps / 606kbps
  • 接続品質優先 — 330kbps / 303kbps
  • 最適音質（Adaptive Bitrate） — 環境に応じて自動調整

注意：この設定はBluetooth送信時のみ有効です。

9. Bluetooth audio codec LHDC（LHDC再生品質）

LHDCコーデック使用時の再生品質モードを切り替えます。

  • 音質優先 — 900kbps
  • バランス — 500kbps / 560kbps
  • 接続品質優先 — 400kbps
  • 低遅延優先 — 256kbps
  • 最適音質（Adaptive Bitrate） — 環境に応じて自動調整

注意：この設定はBluetooth送信時のみ有効です。

10. Bluetooth volume adjustment mode（Bluetooth音量調整モード）

Bluetooth送信時の音量調整方式を選択します。

  • Sync Bluetooth volume（同期モード） — M27と接続先デバイスの音量が連動します。接続先デバイスが音量同期に対応している必要があります。AirPods等で音量が小さくなる問題を解決できます。
  • Indep Bluetooth volume（独立モード） — 送信側と受信側の音量調整が独立して動作します。

11. 3.5mm/4.4mm output mode（出力モード切替）

3.5mm端子および4.4mm端子の出力モードを切り替えます。

  • 3.5mm端子 — PO（ヘッドホン出力、デフォルト）/ LO（ライン出力）
  • 4.4mm端子 — BAL（バランス出力、デフォルト）/ BAL.LO（バランスライン出力）

警告：ライン出力に切り替えると音量が最大になります。切り替える前に必ずヘッドホンを外してください。

12. SPDIF OUT（SPDIF出力設定）

DSDトラックの同軸出力時の変換方式を設定します。

  • DoP（デフォルト） — PCMおよびDSD信号をネイティブ出力します（DSD128まで対応）。
  • D2P — PCM信号はそのまま出力し、DSD信号はPCMに変換して出力します。DSD64/DSD128は88.2kHz PCMに、DSD256は176.4kHz PCMに変換されます。

注意：DoP非対応のデバイスを接続する場合のみ、D2Pを選択してください。

13. Lineout volume adjustable（ライン出力音量調整）

有効にすると、LO（ライン出力）時の音量をボタンで調整できるようになります。シングルエンド（3.5mm）およびバランス（4.4mm）のライン出力の両方に適用されます。

14. Adjustable system USB output volume（USBオーディオ出力音量調整）

有効にすると、USBオーディオの出力音量をM27本体から直接調整できるようになります。

15. Second harmonic adjustment（2次高調波調整）

真空管アンプ特有の歪み特性を再現する機能です。元の信号には存在しない偶数次高調波（2次高調波）を意図的に付加します。

  • Level 0 — 最小高調波、最低歪み
  • Level 1 — 中程度の高調波、中程度の歪み
  • Level 2 — 最大高調波、最高歪み

設定変更は即座に反映されます。

16. Equalizer（イコライザー / PEQ）

31バンドのグローバルPEQ（パラメトリックイコライザー）機能です。FiiO Musicおよびサードパーティアプリに対してEQの重ね掛けが可能です。

  • 192kHzを超えるソースはSRC処理されます。
  • 優先順位: All to DSD > PEQ > MQA
  • Android / Pure Music / AirPlay / Roon Readyモードのアナログ出力のみ有効です。
  • Bluetooth受信モードおよびUSB DACモードでは無効です。

17. Auto pause without headphones（ヘッドホン未接続時の自動一時停止）

有効にすると、AndroidモードおよびPure Musicモードで動作します。ヘッドホンが接続されていない状態で音楽再生中の場合、2分後に自動的に再生が一時停止されます。

Display（ディスプレイ）

1. Sample rate display（サンプリングレート表示）

オンにすると、再生中の楽曲のサンプリングレートがステータスバーに表示されます。

2. Dark theme（ダークテーマ）

画面をダークモードに切り替えます。暗い場所での使用時に目の負担を軽減します。

3. Screen timeout（画面タイムアウト）

無操作時に画面が自動消灯するまでの時間を設定します。

4. Font size（フォントサイズ）

画面のフォントサイズを変更します。

5. Brightness level（明るさ）

M27の画面の明るさを調整します。

6. Indicator light control（インジケーターライト制御）

側面LEDライトの制御設定です。

  • 明るさ — 1〜9段階で調整可能
  • 動作モード — オフ / 常時点灯 / パルス / リズム（再生中の音楽に合わせて点滅）
  • 光の色 — オーディオフォーマットに基づく（デフォルト: 青）/ 固定色（赤 / 青 / シアン / 紫 / 黄 / 白 / 緑 / オレンジ / レインボー）
  • ディスプレイ連動 — 画面のオン/オフに同期

7. Night light（ナイトライト）

ナイトライトモードでは画面が琥珀色に調整されます。カスタム時間設定、または日没から日の出まで自動有効化が可能です。

8. Display size（表示サイズ）

画面上の項目を大きくまたは小さく表示します。

9. Lock screen（ロック画面）

画面ロック時に表示するコンテンツを設定します。

10. Auto-rotate screen（自動画面回転）

M27には重力センサーが内蔵されており、自動画面回転に対応しています。ドロップダウンメニューまたはSettings（設定）→ Display（ディスプレイ）から有効にできます。

Global（グローバル）

1. System navigation（システムナビゲーション）

ナビゲーション方式を選択します。

  • Gesture（ジェスチャー） — スワイプ操作で戻る、ホーム、タスク切り替えを行います。
  • 2-button（2ボタン） — ホームボタンと戻るボタンの2つで操作します。
  • 3-button（3ボタン） — 従来の戻る、ホーム、タスク切り替えの3ボタンで操作します。

2. Tape mode theme（テープモードテーマ）

テープモードで使用するテーマを選択します。詳細は「テープモード」を参照してください。

3. Multifunction button（マルチファンクションボタン）

マルチファンクションボタンに割り当てる機能を変更できます。

プレーヤーモード時の選択肢:

  • お気に入りに追加（デフォルト）
  • ランダム再生
  • 再生 / 一時停止
  • 再生中の曲を削除
  • フィルター切替
  • イコライザー切替
  • USB DACモードに入る
  • Bluetooth受信モードに入る
  • 画面ロック / 復帰
  • 機能なし

注意：Bluetooth受信モードおよびUSB DACモード時は「機能なし」に強制設定されます。AndroidモードおよびPure Musicモードのみ対応です。一部の機能はFiiO Music使用時のみ有効です。

4. Power-off timers（電源オフタイマー）

  • Sleep（スリープ） — 設定時間経過後に自動的に電源がオフになります。0〜120の任意の整数値（分）で設定可能です。
  • Idle poweroff（アイドル自動電源オフ） — 操作や再生がない状態で設定時間に到達すると自動的に電源がオフになります。5分〜24時間で設定可能です。操作や再生があるとタイマーがリセットされます。OFFに設定すると無効になります。

5. HOLD switch settings（HOLDスイッチ設定）

HOLDスイッチをオンにした際に無効化するボタン、ノブ、タッチスクリーンを選択できます。Settings（設定）→ Global（グローバル）→ HOLD Switch（HOLDスイッチ）から設定します。

6. Volume setting（音量設定）

  • System Maximum volume（システム最大音量） — この設定値を超える音量にはなりません。
  • Power-up default setting（起動時デフォルト音量） — 電源オン時の初期音量を設定します。

注意：システム最大音量は起動時デフォルト音量より優先されます。デフォルト音量がシステム最大音量を超えて設定されている場合、システム最大音量が適用されます。

7. Double tap to wakeup（ダブルタップで復帰）

有効にすると、画面オフまたはスリープ状態の時に画面をダブルタップして復帰できます。

Apps（アプリ）

インストール済みアプリの管理、権限の確認、デフォルトアプリの設定を行います。

  • アプリの権限の変更
  • アプリの強制終了
  • アプリの無効化・アンインストール

System（システム）

Languages & input（言語と入力）

初回起動時にシステム表示言語を選択できます。後から変更する場合は以下の手順で行います。

  1. Settings（設定）→ System（システム）→ Languages & input（言語と入力）→ Languages を開きます。
  2. Languages ページで「Add a language」をタップします。
  3. 追加したい言語を選択します。
  4. 目的の言語の右側にある「＝」アイコンを長押しし、一番上までドラッグします。

注意：FiiO Musicアプリの言語はデフォルトでシステムの言語に従います。FiiO Musicアプリ → ME → Languages から個別に変更することも可能です。

Date & time（日付と時刻）

日付と時刻を設定します。Wi-Fi接続時に自動設定できます。

Reset options（リセット）

ネットワーク設定のリセット、アプリ設定のリセット、工場出荷時リセットを実行できます。

  • Reset WiFi, mobile & Bluetooth — すべてのネットワーク設定（Wi-Fi、Bluetooth）をリセットします。
  • Reset app preferences — すべてのアプリ設定をリセットします。アプリのデータは削除されません。
  • Erase all data（factory reset） — 工場出荷時リセットを実行します。内部ストレージのすべてのデータが消去されます。詳細は「工場出荷時リセットの詳細手順」を参照してください。

About Device（デバイス情報）

モデル名、ファームウェアバージョン、シリアル番号などの情報を確認できます。オンラインアップデートとローカルアップデートもこの画面から実行します。

FiiO Music

FiiO Musicは、M27にプリインストールされている高機能音楽再生アプリです。

ホームページ

FiiO Musicを起動するとホームページが表示されます。上部のタブで以下のカテゴリーを切り替えられます。

  • All Songs（すべての曲） — ライブラリのすべての楽曲を表示します。
  • Artist（アーティスト） — アーティスト別に分類します。
  • Album（アルバム） — アルバム別に分類します。
  • Genre（ジャンル） — ジャンル別に分類します。
  • Folder（フォルダー） — フォルダー構造で表示します。

画面右側のレターバーを指でスライドすると、画面中央にアルファベットが大きく表示されます。対応する頭文字の曲に、すばやくジャンプできます。

隠しメニュー

各曲の右端にある⫶アイコンをタップすると、隠しメニューが表示されます。プレイリストへの追加、タグ編集、ファイル情報の表示などの操作が可能です。

並べ替え

曲の並べ替え方法は以下から選択できます。

  • Sort by date added（追加日順）
  • Sort by filename（ファイル名順）
  • Sort by first letter（頭文字順）
  • Sort by year（年代順）

一括管理

一括管理モードでは、複数の曲を選択して一度にプレイリストに追加、削除などの操作ができます。

再生画面

画面下部のアルバムカバーをタップすると、再生画面に切り替わります。

画面切り替え

再生画面のアルバムカバー中央をタップすると、アルバムカバーと歌詞の表示が順に切り替わります。左右にスワイプすると前の曲/次の曲に切り替えられます。

再生プログレス

アルバムカバーの下のプログレスバーを左右にスライドすると、再生位置をすばやく調整できます。

カバー表示スタイル

再生画面のカバー表示は、スクエアカバーとラージカバーの2種類から選択できます。

イコライザー

  • プリセットEQ — 画面下部を左右にスワイプしてプリセットを切り替えます。
  • カスタムEQ — 各周波数バンドを上下にドラッグしてEQカーブを調整します。バンドページを左右に切り替えると異なる周波数帯域を調整できます。

VUメーター表示

再生画面でアルバムカバーをタップすると、VUメーター表示ページに切り替わります。dB閾値とダイナミックポインターが表示されます。0dBの赤い閾値に近づくと、オーディオの歪みを示します。

スペクトラム表示

再生画面にスペクトラムモードが搭載されています。3種類のスペクトラムエフェクトから選択できます。フルスクリーン表示とオリジナル表示を切り替えられます。

有効化: FiiO Musicアプリ → Settings（設定）→ Theme（テーマ）→ Now playing page（再生画面）→ Spectrum display（スペクトラム表示）を選択します。

テープモード

M27にはカセットテープ風のビジュアルで音楽を楽しめるテープモードが搭載されています。複数のテーマから選択できます。

テーマの種類

Spectrum theme（スペクトラムテーマ）: 3種類

VU meter theme（VUメーターテーマ）: 8種類

テープモードの起動方法

Settings（設定）→ Global（グローバル）→ Tape Mode Theme（テープモードテーマ）からテーマを選択した後、以下の方法でテープモードを起動できます。

  1. テーマ選択ページの下部にある「Start Now」ボタンをタップします。
  2. 再生画面からテープモードアイコンをタップします。
  3. クイック設定パネルからテープモードを起動します。

よくある質問

電源が入らない場合

バッテリーが完全に放電している可能性があります。充電器に接続して30分以上充電してからお試しください。それでも起動しない場合は、電源ボタンを10秒以上長押しして強制再起動してください。

音が出ない場合

  • 音量が0になっていないか確認してください。
  • ヘッドホンのプラグがしっかり差し込まれているか確認してください。
  • 光/同軸デジタル出力がオンになっていないか確認してください。光/同軸出力がオンの場合、3.5mm端子からアナログ音声は出力されません。
  • USB DACモードやCOAXモードになっていないか確認してください。
  • HOLDスイッチがオンになっていないか確認してください。

microSDカードが認識されない場合

  • カードを一度取り外し、再度挿入してください。
  • M27を再起動してください。
  • カードのファイルシステムがexFATまたはFAT32であることを確認してください。
  • 他のデバイスでカードの動作を確認してください。カード自体の不良の可能性もあります。

充電が遅い場合

急速充電を利用するには、QC4+またはPD3.0対応の充電器をご使用ください。非対応の充電器では充電速度が遅くなります。充電中に使用していると、充電速度が低下する場合があります。

本体が熱くなる場合

高ゲインでの長時間再生、All to DSD機能の使用、充電しながらの使用などで本体温度が上昇する場合があります。これは正常な動作です。ただし、異常に高温になった場合は使用を中止してください。

Wi-Fiが不安定な場合

  • ルーターとの距離を確認してください。
  • 2.4GHz帯と5GHz帯を切り替えてお試しください。
  • Settings（設定）→ Wi-Fiから、ネットワークを「削除」して再接続してください。

工場出荷時リセットの方法

Settings（設定）→ System（システム）→ Reset options（リセット）→ Erase all data（すべてのデータを消去）を選択します。

警告：工場出荷時リセットを実行すると、内部ストレージのすべてのデータが消去されます。実行前にバックアップを取ってください。microSDカード上のデータは消去されません。

詳細FAQ

Google Playのアンインストール方法

Settings（設定）→ Apps（アプリ）→ All apps（すべてのアプリ）で「Google Play Services」を選択し、「UNINSTALL」をタップします。

注意：将来復元が必要な場合は、工場出荷時リセットで復元できます。

ファイル転送方法

USBケーブルでPC/Macに接続します。M27画面に表示されるUSBモード選択で「File Transfer」を選択します。

macOSの場合はAndroid File Transferアプリが必要です。

OTG機能の使い方

USB OTGケーブルで外部USBストレージを接続します。FiiO Musicアプリのフォルダーブラウズから、USBストレージのファイルを再生できます。

USB DACの使い方（スマートフォン接続）

  1. M27をUSB DACモードに切り替えます。
  2. OTGケーブルでスマートフォンとM27（USB 3.0ポート）を接続します。
  3. ヘッドホンをM27に接続して再生します。

iOS/iPadOSの場合: M27のUSB DACページ設定で「iOSデバイスデコード」を有効化してください。

Androidの場合: 向き付きOTGケーブルを使用します。一部のデバイスでは開発者オプションの「USBオーディオ出力」の有効化が必要です。

注意：USB DACモードではBluetooth送信は使用できません。

AirPlayの使い方

  1. M27をAirPlayモードに切り替えます。
  2. M27とAppleデバイスを同じWi-Fiに接続します。
  3. Appleデバイスのコントロールセンターで「FiiO M27」を選択します。

注意：AirPlay接続には安定したWi-Fi環境が必要です。M27がスリープ状態でも接続可能です。

Roon Readyの使い方

  1. M27を起動し、AndroidモードまたはPure Musicモードに設定します。
  2. M27とRoonサーバーを同じネットワークに接続します。
  3. FiiO MusicアプリがRoon Ready出力として自動検出されます。

注意：Roonのサブスクリプションが必要です。

Bluetoothコーデックの確認と切替方法

対応コーデック: LHDC、LDAC、aptX Adaptive、aptX HD、aptX、AAC、SBC

Settings（設定）→ Audio（オーディオ）→ Bluetooth Audio Encoder で切り替えできます。

注意：コーデック変更後はBluetoothを切断し、再接続する必要があります。

Bluetoothイヤホンに接続できない場合

初回接続時は、イヤホンがペアリング状態であることを確認してください。再接続状態では初回接続に失敗します。

AirPods接続時に音量が小さい場合

  1. Settings（設定）→ Audio（オーディオ）→ Bluetooth volume adjustment mode（Bluetooth音量調整モード）を開きます。
  2. 「Bluetooth音量同期」に切り替えます。
  3. Bluetoothを切断し、再接続します。

デスクトップモードの詳細

デスクトップモードの詳細は「デスクトップモード」セクションを参照してください。

USB 3.0ポートで急速充電アダプターを接続すると、スーパーハイゲインが利用可能になります。

スーパーハイゲインモードの有効化方法

  1. デスクトップモードスイッチをONにします。
  2. USB 3.0ポート（POWER IN）にPD3.0対応急速充電アダプター（9V/3A、12V/1.67A以上）を接続します。
  3. クイック設定パネルまたはSettings（設定）→ Audio（オーディオ）→ Gain（ゲイン）で「Ultra high」を選択します。

注意：急速充電のPOWER INポートに接続し、デスクトップモードをONにするとプロンプトが表示されます。

FiiO Cast リモートコントロールの使い方

  1. FiiO Controlアプリをスマートフォンにインストールします。
  2. M27とスマートフォンを同じネットワークに接続します。
  3. M27でFiiO Castアプリを起動し、「Start」をタップします。
  4. スマートフォンのFiiO Controlアプリで Add New Device → Player → M27のIPアドレスを検索して接続します（またはQRコードで接続）。

トラブルシューティング: 接続できない場合は、ネットワーク設定、バックグラウンドサービスの状態、IPアドレスを確認してください。

DK1 PLUSクーリングスタンドについて

付属のDK1 PLUSには以下の機能があります。

  • 外部電源接続時にファンでM27を冷却します。
  • 縦置き・横置き両対応のブラケットです。
  • TYPE-C接続でM27を充電できます。USB HOSTポート2つからUSB Audio出力が可能です。

注意：DK1 PLUSはUltra High Gainモードの使用には不要です。M27単体でも急速充電アダプター直結で使用可能です。USB 3.0ポート経由での接続を推奨します。接続前にM27底面の光/同軸ダストプラグを取り外してください。

M27のサウンドチューニングについて

M27はリファレンスグレードプレーヤーとして、ニュートラルで堅実な音質を目指しています。

  • 高精度クリスタルオシレーター、大容量FPGA、40Wパワーシステムを搭載しています。
  • ES9039SPRO DACの新ハイパーストリーム技術により、音の密度と厚みを実現しています。
  • 高品質オペアンプ、フィルムコンデンサ、ELNAオーディオグレード電解コンデンサを採用しています。
  • 多段ハードウェアゲインと電源レギュレーションを搭載しています。

再生開始時に「カチッ」という音がする

内部リレーの動作音です。再生開始時にリレーが作動し、クリック音が発生します。これは正常な現象であり、故障ではありません。

安全上のご注意

ご使用前に以下の安全上のご注意をよくお読みになり、正しくお使いください。

危険：長時間大音量で聴かないでください。聴覚を損傷するおそれがあります。音量は周囲の音が聞こえる程度に設定してください。

危険：自動車や自転車の運転中にヘッドホンを使用しないでください。事故の原因となります。

警告：本製品を分解・改造しないでください。感電や故障の原因となります。修理はFiiO正規サービスにご依頼ください。

警告：水や液体に浸さないでください。本製品は防水仕様ではありません。

警告：直射日光の当たる場所や高温の場所に放置しないでください。バッテリーの劣化や故障の原因となります。

警告：付属品以外の充電器を使用する場合は、出力仕様が本製品に適合していることを確認してください。不適切な充電器の使用は故障の原因となります。

注意：本製品のバッテリーを長期間使用しない場合は、50%程度まで充電した状態で涼しい場所に保管してください。完全放電状態で長期保管すると、バッテリーの劣化が促進されます。

バッテリーに関する注意事項

  • 本製品にはリチウムポリマーバッテリーが内蔵されています。
  • 火中に投じないでください。爆発のおそれがあります。
  • バッテリーを直接交換しないでください。内蔵バッテリーの交換はFiiO正規サービスにご依頼ください。

聴覚保護について

大音量で長時間聴き続けると、永久的な聴覚障害を引き起こすおそれがあります。以下のことを守ってください。

  • 適切な音量で使用してください。耳鳴りがする場合は音量を下げてください。
  • 長時間の連続使用を避け、定期的に休憩してください。
  • 使用開始時は音量を小さくしてから、徐々に上げてください。
