外観ガイド — ボタン・端子の説明

上面

  • 3.5mmヘッドホン出力 / ライン出力 — シングルエンドのヘッドホン出力端子です。ライン出力としても使用できます。
  • 4.4mmバランスヘッドホン出力 / バランスライン出力 — バランス接続用のヘッドホン出力端子です。バランスライン出力としても使用できます。
  • 6.35mmヘッドホン出力 — デスクトップモード時に使用する標準サイズのヘッドホン端子です。
  • 電源/ボリュームノブ — 回転して音量を調整します。押すと画面のオン/オフを切り替えます。長押しで電源オン/オフを行います。

左側面

  • マルチファンクションボタン — 短押しでFiiO Musicアプリに切り替わります。長押しでモード選択メニューを表示します。
  • 前の曲ボタン — 短押しで前のトラックに戻ります。長押しで巻き戻しします。
  • 再生/一時停止ボタン — 再生と一時停止を切り替えます。
  • 次の曲ボタン — 短押しで次のトラックに進みます。長押しで早送りします。

右側面

  • アンテナ — Wi-FiおよびBluetoothの受信感度を向上させます。
  • アンビエントライト — 側面のLEDライトです。再生状態やサンプリングレートに応じて色が変わります。

底面

  • 光（オプティカル）出力 — 光デジタル信号を出力します。
  • USB Type-C（充電/電源） — 充電および電源供給に使用します。
  • HOLDスイッチ — 物理ボタンの誤操作を防止するロックスイッチです。
  • USB Type-C（データ） — データ転送、USB DACモード、USB OTG接続に使用します。
  • MicroSDカードスロット — 最大2TBのmicroSDカードに対応します。
  • USB HOST — USB DACやUSBメモリなどの外部USBデバイスを接続します。
  • 同軸（コアキシャル）入出力（COAX IN/OUT） — 同軸デジタル信号の入力・出力に使用します。
  • デスクトップモードスイッチ — デスクトップモードのオン/オフを物理的に切り替えます。

USB Type-Cポートの機能比較

M27には2つのUSB Type-Cポートがあります。用途が異なるため、接続時にご注意ください。

機能 USB Type-C（データ） USB Type-C（充電/電源）
充電 非対応 対応（QC4+/PD3.0）
データ転送 対応 非対応
USB DACモード 対応 非対応
USB OTG 対応 非対応
DC電源入力（デスクトップモード） 非対応 対応

出力端子の優先順位

M27では複数の出力端子を同時に使用することはできません。出力の優先順位は以下のとおりです。

  1. 6.35mmヘッドホン出力（デスクトップモード時のみ、最優先）
  2. 4.4mmバランスヘッドホン出力
  3. 3.5mmヘッドホン出力

上位の端子にプラグが接続されている場合、下位の端子からは音声が出力されません。

注意：ライン出力として使用する場合は、Settings（設定）→ Audio（オーディオ）で出力モードをLine Out（ライン出力）に切り替えてください。

デスクトップモードスイッチの詳細

底面のデスクトップモードスイッチでポータブルモードとデスクトップモードを切り替えます。各モードで使用できる機能は以下のとおりです。

機能 ポータブルモード デスクトップモード
3.5mmヘッドホン出力 対応 対応
4.4mmバランス出力 対応 対応
6.35mmヘッドホン出力 非対応 対応
USB HOST 非対応 対応
COAX IN（同軸入力） 非対応 対応
COAX OUT（同軸出力） 対応 対応
バッテリー駆動 対応 非対応（常時電源接続が必要）

製品概要

FiiO M27はフラッグシップクラスのAndroidベースデジタルオーディオプレーヤー（DAP）です。ESS ES9038PRO D/Aコンバーター（DAC）を2基搭載し、THX AAA-788+ヘッドホンアンプを2基搭載しています。

ポータブルモードとデスクトップモードの2つの使い方ができます。デスクトップモードでは6.35mmヘッドホン出力やUSB HOST、同軸入力が利用可能になります。

主な特長

  • ESS ES9038PRO × 2基によるデュアルDAC構成
  • THX AAA-788+ × 2基によるヘッドホンアンプ
  • 4.4mmバランス出力で最大1500mW+1500mW（32Ω負荷時）
  • 5.0インチIPS液晶（1080×1920）
  • Qualcomm Snapdragon 665プロセッサー
  • 4GB RAM / 64GB内部ストレージ
  • 9800mAh大容量バッテリー（最大約15時間再生）
  • Bluetooth 5.1（LDAC / aptX Adaptive / aptX HD対応）
  • Roon Ready / AirPlay / DLNA対応
  • MQA 8×アンフォールド対応

付属品

  • FiiO M27本体
  • USB Type-Cケーブル
  • 保護フィルム（貼付済み）
  • クイックスタートガイド
  • 保証書

クイックスタート

充電

初めて使用する際は、十分に充電してください。底面のUSB Type-C（充電/電源）ポートに充電器を接続します。

  • QC4+ / PD3.0対応の充電器で最大27Wの急速充電が可能です。
  • フル充電まで約3時間です（急速充電器使用時）。

電源オン/オフ

  1. 電源オン — 電源/ボリュームノブを長押しします。FiiOのロゴが表示されたら起動中です。
  2. 電源オフ — 電源/ボリュームノブを長押しし、表示されたメニューから「Power off」（電源オフ）を選択します。
  3. 画面のオン/オフ — 電源/ボリュームノブを短く押します。

microSDカードの挿入

  1. 底面のmicroSDカードスロットに、microSDカードをカチッと音がするまで差し込みます。
  2. M27が自動的にカードを認識します。

注意：microSDカードは電源オフ時に挿入・取り外しすることを推奨します。使用中に取り外す場合はSettings（設定）→ Storage（ストレージ）から安全な取り外しを実行してください。

ステータスバーと表示画面

画面上部のステータスバーには、M27の動作状態が表示されます。

画面上部から下にスワイプすると、クイック設定パネルが表示されます。以下の機能に素早くアクセスできます。

  • Internet — Wi-Fiのオン/オフを切り替えます。
  • Bluetooth — Bluetoothのオン/オフを切り替えます。
  • Mode Choose（モード選択） — 6つの動作モードを切り替えます。
  • Gain（ゲイン） — ヘッドホン出力のゲインをLow / High / Ultra Highの3段階で切り替えます。
  • PO — 3.5mmヘッドホン出力をオン/オフします。
  • BAL.PO — 4.4mmバランスヘッドホン出力をオン/オフします。
  • OPT/COAX — 光/同軸デジタル出力をオン/オフします。
  • Auto-rotate（自動回転） — 画面の自動回転を切り替えます。

6つの動作モード

FiiO M27には6つの独立した動作モードがあります。Mode Choose（モード選択）から切り替えできます。

Android Mode（Androidモード）

標準的な動作モードです。すべての機能とサードパーティアプリが利用できます。ストリーミングサービスやファイル管理など、フル機能のAndroid環境として使用します。

Pure Music Mode（Pure Musicモード）

音楽再生に特化したモードです。不要なバックグラウンドプロセスを停止し、オーディオ品質を最大限に高めます。FiiO Musicアプリのみが動作します。

Pure Musicモードではバッテリー駆動時間も延長されます。

AirPlay Mode（AirPlayモード）

M27をAirPlayレシーバーとして動作させるモードです。iPhone、iPad、Macからワイヤレスで音楽を送信して再生できます。

  1. M27とAppleデバイスを同じWi-Fiネットワークに接続します。
  2. Mode ChooseからAirPlayモードを選択します。
  3. Appleデバイスのコントロールセンターで、AirPlay出力先として「FiiO M27」を選択します。
  4. Appleデバイスで音楽を再生すると、M27から音が出力されます。

USB DAC Mode（USB DACモード）

M27をUSB DACとして使用するモードです。パソコンやスマートフォンに接続して、M27の高品質なDACとアンプで音楽を再生します。

Windowsの場合

  1. Mode ChooseからUSB DAC Modeを選択します。
  2. 底面のUSB Type-C（データ）ポートでパソコンに接続します。
  3. FiiO USB DACドライバーをインストールします（初回のみ）。
  4. Windowsのサウンド設定で「FiiO M27」を出力デバイスとして選択します。

macOS / Linuxの場合

  1. Mode ChooseからUSB DAC Modeを選択します。
  2. 底面のUSB Type-C（データ）ポートでパソコンに接続します。
  3. システムのサウンド設定で「FiiO M27」を出力デバイスとして選択します。

macOSおよびLinuxではドライバーのインストールは不要です。

対応フォーマット（USB DACモード）

  • PCM: 最大384kHz/32bit
  • DSD: DSD64/128/256（ネイティブ / DoP）
  • MQA対応

Bluetooth Receiving Mode（Bluetooth受信モード）

M27をBluetoothレシーバーとして使用するモードです。スマートフォンなどからBluetoothで音楽を受信して再生します。

  1. Mode ChooseからBluetooth Receiving Modeを選択します。
  2. スマートフォンのBluetooth設定で「FiiO M27」を検索してペアリングします。
  3. スマートフォンで音楽を再生すると、M27から音が出力されます。

対応Bluetoothコーデック（受信）: SBC、AAC、LDAC

COAX Mode（COAXモード）

M27を同軸DACとして使用するモードです。外部機器から同軸デジタル信号を入力し、M27のDACとアンプで再生します。

  1. Mode ChooseからCOAX modeを選択します。
  2. 底面の同軸入力端子に同軸デジタルケーブルを接続します。
  3. 外部機器で再生を開始すると、M27から音が出力されます。

よく使う機能の説明

スクリーンショット

M27でスクリーンショットを撮影できます。

操作方法: 電源ボタンと前の曲ボタンを同時に押します。画面が一瞬フラッシュし、スクリーンショットが保存されます。

保存先は内部ストレージのPictures/Screenshotsフォルダーです。

USBオーディオデジタル出力

M27のUSB HOST端子から外部USB DACにデジタルオーディオ信号を出力できます。詳しい手順は「USBオーディオ出力」を参照してください。

FiiO Link — Bluetooth制御

FiiO Linkを使用すると、スマートフォンからBluetooth経由でM27をリモートコントロールできます。

  1. スマートフォンにFiiO Musicアプリをインストールします。
  2. M27のFiiO MusicアプリでFiiO Link機能を有効にします（設定画面から「FiiO Link」をオン）。
  3. M27のBluetooth設定でBluetoothをオンにします。
  4. スマートフォンのFiiO MusicアプリでFiiO Linkを開き、M27を選択して接続します。
  5. 接続後、スマートフォンからM27の音楽ライブラリの閲覧や再生コントロールができます。

FiiO Link — Wi-Fi制御

Wi-Fi経由でFiiO Linkを使用することもできます。Bluetoothよりも安定した接続で操作できます。

  1. M27とスマートフォンを同じWi-Fiネットワークに接続します。
  2. M27のFiiO MusicアプリでFiiO Link機能を有効にします。
  3. スマートフォンのFiiO MusicアプリでFiiO Linkを開き、Wi-Fi経由でM27を選択して接続します。
  4. 接続後、スマートフォンからM27をリモートコントロールできます。

DLNA機能

FiiO M27はDLNA（Digital Living Network Alliance）に対応しています。同じWi-Fiネットワーク上にあるDLNA対応デバイスと音楽ファイルを共有・再生できます。

M27をDLNAレンダラーとして使用する

パソコンやNASなどのDLNAサーバーから音楽をM27に送信して再生できます。

  1. M27とDLNAサーバーを同じWi-Fiネットワークに接続します。
  2. FiiO Musicアプリを起動します。
  3. DLNAコントローラーアプリ（パソコンのメディアプレーヤーなど）からM27を出力先として選択します。
  4. DLNAコントローラーで音楽を選択して再生します。M27から音が出力されます。

M27でDLNAサーバーの音楽をブラウズする

M27からDLNAサーバー上の音楽ファイルをブラウズして再生できます。

  1. M27とDLNAサーバーを同じWi-Fiネットワークに接続します。
  2. FiiO Musicアプリのメニューから「DLNA」を選択します。
  3. 表示されたDLNAサーバーを選択し、音楽ファイルをブラウズします。
  4. 再生したい曲を選択します。

Roon Ready

FiiO M27はRoon Readyに対応しています。Roonアプリから高品質なストリーミング再生が可能です。

  1. M27とRoon Coreサーバーを同じネットワークに接続します。
  2. FiiO Musicアプリを起動します。Roon Ready機能は自動的に有効になります。
  3. Roonアプリの Settings（設定）→ Audio でFiiO M27が表示されます。「Enable」をクリックして有効にします。
  4. Roonアプリのゾーン選択でM27を出力先として選択します。
  5. Roonアプリから音楽を再生すると、M27から音が出力されます。

注意：Roon Readyを使用するには、別途RoonのサブスクリプションとRoon Coreサーバーが必要です。

光/同軸出力の使い方

M27は光デジタル出力および同軸デジタル出力に対応しています。外部DACやアンプに接続する際に使用します。

光デジタル出力

  1. 底面の光出力端子に光デジタルケーブルを接続します。
  2. Settings（設定）→ Audio（オーディオ）→ Optical/Coaxial output（光/同軸出力）をオンにします。
  3. FiiO Musicで音楽を再生すると、光デジタル信号が出力されます。

同軸デジタル出力

  1. 底面の同軸出力端子に同軸デジタルケーブルを接続します。
  2. Settings（設定）→ Audio（オーディオ）→ Optical/Coaxial output（光/同軸出力）をオンにします。
  3. FiiO Musicで音楽を再生すると、同軸デジタル信号が出力されます。

注意：光/同軸出力をオンにすると、3.5mm端子からのアナログ出力は無効になります。アナログ出力に戻すには、光/同軸出力をオフにしてください。

DSD信号をS/PDIFで伝送する場合、DoP（DSD over PCM）方式を使用します。Settings（設定）→ Audio（オーディオ）でDoPの設定を変更できます。

デスクトップモード

FiiO M27はポータブルモードとデスクトップモードの2つの形態で使用できます。底面のデスクトップモードスイッチを切り替えることで、モードが変わります。

デスクトップモードの特長

  • 6.35mmヘッドホン出力 — デスクトップモードでのみ使用可能です。高インピーダンスのフルサイズヘッドホンを駆動できます。
  • USB HOST — 外部USB DACやUSBメモリを接続できます。M27をトランスポートとして使用する場合に便利です。
  • COAX IN（同軸入力） — 外部機器から同軸デジタル信号を受信します。
  • 常時電源供給 — USB Type-C（充電/電源）から常時電源を供給しながら使用できます。バッテリー劣化を防ぐバイパス充電機能を搭載しています。

注意：デスクトップモードスイッチの切り替え後、M27が自動的に再起動する場合があります。データの保存を確認してから切り替えてください。

Bluetooth送信設定

M27をBluetoothヘッドホンやBluetoothスピーカーに接続する場合の設定です。

  1. Settings（設定）→ Bluetooth メニューを開き、Bluetoothをオンにします。
  2. 接続先のBluetoothデバイス（イヤホン、スピーカーなど）をペアリングモードにします。
  3. M27のBluetooth設定画面に検出されたデバイスが表示されます。デバイスをタップしてペアリングします。
  4. 「Connected」（接続済み）と表示されたら接続完了です。

対応Bluetoothコーデック（送信）: SBC、AAC、aptX、aptX HD、aptX Adaptive、LDAC

注意：Bluetoothコーデックは接続先のデバイスが対応しているコーデックに自動的に切り替わります。高音質コーデックを使用するには、接続先のデバイスも同じコーデックに対応している必要があります。

USBオーディオ出力

M27のUSB HOST端子に外部USB DACを接続して、デジタルオーディオ信号を出力できます。M27をトランスポート（デジタル音源送出装置）として使用する場合に有効です。

  1. デスクトップモードスイッチをオンにします。
  2. 底面のUSB HOST端子に外部USB DACを接続します。
  3. Settings（設定）→ Audio（オーディオ）→ USB Audio Output（USBオーディオ出力）をオンにします。
  4. FiiO Musicで音楽を再生すると、外部USB DACから音が出力されます。

注意：FiiO Music以外のサードパーティアプリでは、USBオーディオ出力が正常に動作しない場合があります。ビットパーフェクト出力にはFiiO Musicの使用を推奨します。

ファームウェアアップデート

M27はオンラインアップデートとローカルアップデートの2つの方法でファームウェアを更新できます。

オンラインアップデート（OTA）

  1. M27をWi-Fiに接続します。
  2. Settings（設定）→ About Device（デバイス情報）→ Online Upgrade（オンラインアップグレード）を選択します。
  3. 新しいファームウェアが利用可能な場合、アップデートの案内が表示されます。
  4. 「Download and Install」（ダウンロードしてインストール）を選択して実行します。

ローカルアップデート

  1. FiiO公式サイトからファームウェアファイルをダウンロードします。
  2. ファームウェアファイルをM27の内部ストレージのルート（最上位フォルダー）にコピーします。
  3. Settings（設定）→ About Device（デバイス情報）→ Local Upgrade（ローカルアップグレード）を選択します。
  4. ファームウェアファイルを選択して、アップデートを実行します。

警告：ファームウェアアップデート中は電源を切らないでください。バッテリー残量が30%以上あることを確認してから実行してください。アップデートの失敗は故障の原因となります。

ファクトリーリストアモードが表示されてファームウェアが検出されない場合は、インストールパッケージが破損している可能性があります。電源ボタンを押して「reboot」（再起動）を選択し、ファームウェアファイルを再度ダウンロードしてお試しください。

強制再起動

M27がフリーズして操作を受け付けない場合、強制再起動を行います。

操作方法: 電源ボタンを10秒以上長押しします。M27が強制的に再起動します。

注意：強制再起動を行うと、保存されていないデータが失われる可能性があります。通常の再起動ができない場合にのみ使用してください。

リカバリーモード

M27が正常に起動しない場合や、工場出荷時リセットが必要な場合にリカバリーモードを使用します。

操作方法: 電源オフの状態で、電源ボタンとボリュームノブを同時に長押しします。リカバリーモード画面が表示されます。

リカバリーモードでは以下の操作が可能です。

  • Reboot system now（再起動） — 通常モードで再起動します。
  • Wipe data/factory reset（データ消去/工場出荷時リセット） — すべてのユーザーデータを消去し、工場出荷時の状態に戻します。
  • Apply update from storage（ストレージからアップデート適用） — 内部ストレージまたはmicroSDカードに保存されたファームウェアファイルを適用します。

警告：「Wipe data/factory reset」を実行すると、内部ストレージのすべてのデータが消去されます。必要なデータのバックアップを事前に行ってください。

工場出荷時リセットの詳細手順

工場出荷時リセットは以下の2つの方法で実行できます。

方法1: Settings（設定）から実行

  1. Settings（設定）→ System（システム）→ Reset options（リセット）を開きます。
  2. 「Erase all data (factory reset)」（すべてのデータを消去）を選択します。
  3. 確認画面で「ERASE ALL DATA」を選択して実行します。
  4. M27が自動的に再起動し、工場出荷時の状態に戻ります。

方法2: リカバリーモードから実行

  1. 上記の手順でリカバリーモードに入ります。
  2. 「Wipe data/factory reset」を選択します。
  3. 確認画面で「Yes」を選択します。
  4. 完了後、「Reboot system now」を選択して再起動します。

サードパーティアプリのインストール

FiiO M27にはGoogle Playストアが搭載されていません。サードパーティアプリのインストールにはAPKファイルを使用します。

  1. パソコンでAPKファイルをダウンロードします。
  2. M27をUSBケーブルでパソコンに接続します。
  3. パソコンからM27の内部ストレージにAPKファイルをコピーします。
  4. M27のファイルマネージャーアプリでAPKファイルを開き、インストールします。

注意：初回のAPKインストール時に「不明なソースからのインストールを許可しますか？」という確認が表示されます。「許可」を選択してインストールを続行してください。

FiiO独自のアプリストアも利用できます。ホーム画面の「FiiO Market」アプリから、FiiOが検証済みのアプリをダウンロードできます。

システム設定

Settings（設定）アプリから、M27の各種設定を変更できます。設定メニューは以下のカテゴリーで構成されています。

Wi-Fi

VPN、テザリングなどのネットワーク関連の追加設定です。

Audio（オーディオ）

FiiO M27のオーディオ専用設定です。

1. Optical/Coaxial output（光/同軸出力）

光出力または同軸出力を使用する場合は、このスイッチをオンにしてください。

2. Channel balance（左右バランス）

左右チャンネルの音量バランスを調整します。通常は中央のままで使用してください。

3. Gain（ゲイン）

ヘッドホン出力のゲインを3段階で切り替えます。

  • Low（低ゲイン） — 高感度のイヤホンやIEMに適しています。
  • High（高ゲイン） — 一般的なヘッドホンに適しています。
  • Ultra High（超高ゲイン） — 高インピーダンスや低感度のヘッドホンに適しています。

警告：ゲインを切り替える際は、一度音量を下げてから行ってください。突然の大音量により聴覚を損傷するおそれがあります。

4. All to DSD

PCM音源をリアルタイムでDSDに変換して再生します。DSD64、DSD128、DSD256から変換先を選択できます。

注意：All to DSD機能を有効にすると、CPU負荷が増加し、バッテリー消費と本体温度が上昇します。

5. Digital Filter（DACフィルター）

DACのデジタルフィルターを選択できます。音質の微細な違いを楽しめます。

  • Sharp Roll-Off
  • Slow Roll-Off
  • Short Delay Sharp Roll-Off
  • Short Delay Slow Roll-Off
  • Super Slow Roll-Off

6. Volume Steps（ボリュームステップ）

音量調整の段階数を選択します。120段階（細かい調整）または60段階（標準）から選択できます。

2. Dark theme（ダークテーマ）

3. Screen timeout（画面タイムアウト）

無操作時に画面が自動消灯するまでの時間を設定します。

4. Font size（フォントサイズ）

画面のフォントサイズを変更します。

よくある質問

電源が入らない場合

バッテリーが完全に放電している可能性があります。充電器に接続して30分以上充電してからお試しください。それでも起動しない場合は、電源ボタンを10秒以上長押しして強制再起動してください。

音が出ない場合

  • 音量が0になっていないか確認してください。
  • ヘッドホンのプラグがしっかり差し込まれているか確認してください。
  • 光/同軸デジタル出力（S/PDIF）がオンになっていないか確認してください。光/同軸出力がオンの場合、3.5mm端子からアナログ音声は出力されません。
  • USB DACモードやCOAXモードになっていないか確認してください。
  • HOLDスイッチがオンになっていないか確認してください。

microSDカードが認識されない場合

  • カードを一度取り外し、再度挿入してください。
  • M27を再起動してください。
  • カードのファイルシステムがexFATまたはFAT32であることを確認してください。
  • 他のデバイスでカードの動作を確認してください。カード自体の不良の可能性もあります。

充電が遅い場合

急速充電を利用するには、QC4+またはPD3.0対応の充電器をご使用ください。非対応の充電器では充電速度が遅くなります。充電中に使用していると、充電速度が低下する場合があります。

本体が熱くなる場合

高ゲインでの長時間再生、All to DSD機能の使用、充電しながらの使用などで本体温度が上昇する場合があります。これは正常な動作です。ただし、異常に高温になった場合は使用を中止してください。

Wi-Fiが不安定な場合

  • ルーターとの距離を確認してください。
  • 2.4GHz帯と5GHz帯を切り替えてお試しください。
  • Settings（設定）→ Wi-Fiから、ネットワークを「削除」して再接続してください。

工場出荷時リセットの方法

Settings（設定）→ System（システム）→ Reset options（リセット）→ Erase all data（すべてのデータを消去）を選択します。

警告：工場出荷時リセットを実行すると、内部ストレージのすべてのデータが消去されます。実行前にバックアップを取ってください。microSDカード上のデータは消去されません。

安全上のご注意

ご使用前に以下の安全上のご注意をよくお読みになり、正しくお使いください。

危険：長時間大音量で聴かないでください。聴覚を損傷するおそれがあります。音量は周囲の音が聞こえる程度に設定してください。

危険：自動車や自転車の運転中にヘッドホンを使用しないでください。事故の原因となります。

警告：本製品を分解・改造しないでください。感電や故障の原因となります。修理はFiiO正規サービスにご依頼ください。

警告：水や液体に浸さないでください。本製品は防水仕様ではありません。

警告：直射日光の当たる場所や高温の場所に放置しないでください。バッテリーの劣化や故障の原因となります。

警告：付属品以外の充電器を使用する場合は、出力仕様が本製品に適合していることを確認してください。不適切な充電器の使用は故障の原因となります。

注意：本製品のバッテリーを長期間使用しない場合は、50%程度まで充電した状態で涼しい場所に保管してください。完全放電状態で長期保管すると、バッテリーの劣化が促進されます。

バッテリーに関する注意事項

  • 本製品にはリチウムポリマーバッテリーが内蔵されています。
  • 火中に投じないでください。爆発のおそれがあります。
  • バッテリーを直接交換しないでください。内蔵バッテリーの交換はFiiO正規サービスにご依頼ください。
  • 廃棄する場合は、お住まいの地域の規定に従ってください。

聴覚保護について

大音量で長時間聴き続けると、永久的な聴覚障害を引き起こすおそれがあります。以下のことを守ってください。

  • 適切な音量で使用してください。耳鳴りがする場合は音量を下げてください。
  • 長時間の連続使用を避け、定期的に休憩してください。
  • 使用開始時は音量を小さくしてから、徐々に上げてください。
